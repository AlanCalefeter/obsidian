# **HTML – 03 – Mise en forme**

## **La mise en forme des caractères**

### **La mise en forme logique**

Grâce à ces balises, le navigateur va interpréter le formatage du texte.  
→ utilisées dans environnement non-graphique (ex : pour DOS) → en **utilisées** (?)

- `<blockquote></blockquote>` → indique une citation
- `<code></code>` → indique un code informatique
- `<em></em>` → emphase (affichage : italique, mais auto en sémantique)
- `<strong></strong>` → “gras”
- `<samp></samp>` → affiche caractères de manière littérale

---

### **La mise en forme physique**

Décide de ce que le navigateur affiche :

- `<b></b>` → texte en gras
- `<i></i>` → italique
- `<u></u>` → souligne le texte
- `<small></small>` → police + petite
- `<big></big>` → police + grande
